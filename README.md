# AI-Driven Database Query Engine

## Overview
An AI-powered system that allows users to upload documents (PDF's,PNG's,JPEG's,Images) .The documents are analysed by AI and a schema is generated automatically .
After schema generation tables are automatically generated by AI.
The application combines structured SQL querying with semantic vector search to retrieve accurate information from uploaded documents.

## Features
- Upload PDF, TXT, and image files
- OCR support for scanned documents
- Automatic AI-based schema generation
- MySQL storage
- FAISS vector search
- Hybrid SQL + semantic search
- FastAPI backend
- React frontend
